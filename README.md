# The Archive

## Project Description
The Archive is a web platform designed to help users organize and remember movies, TV series, and books watched or read with friends. The main goal of the project is to provide a single, centralized space where users can save, manage, and revisit cultural content, avoiding the common problem of forgetting what has already been watched or what was planned to be watched or read.

The project is still being developed as a full-stack application, featuring a modern frontend and a dedicated backend responsible for data management. The user interface focuses on simplicity and usability, allowing users to easily browse and maintain their personal and shared archives.

## Main Features
- Create and manage an archive of movies, TV series, and books  
- Track content that has already been watched or read  
- Save content to watch or read in the future  
- Organize and share content with friends  
- Clean and stylized user interface
